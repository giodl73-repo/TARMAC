# TARMAC Open Adoption Guide

## Purpose

TARMAC is public and open to use as a reference model for evidence-gated
aviation-network, delay, capacity, connectivity, resilience, competition, and
access analysis. Its next public milestone is a narrow, source-backed first
cited run.

Public use does not create an engineering study, airspace or procedure design,
environmental review, slot allocation, route award, procurement plan, or FAA,
airport-authority, airline, or ICAO endorsement.

## First Run Path

Use [`first-public-run-worksheet.md`](first-public-run-worksheet.md) to scope
one airport system, regional access problem, route-connectivity question, delay
exposure, hub concentration issue, or ground-access gap.

The first run should prove the source-to-corpus-to-gap workflow, not settle an
airspace, slot, route-award, airport-engineering, or procurement case.

## Contribution Targets

- public source inventories for delay, schedule, capacity, slot, route,
  competition, ground-access, weather, or resilience evidence;
- source/claim corrections for delay, connectivity, market, access, or capacity
  language;
- reviewer notes from aviation planning, airport operations, airline network,
  ground access, resilience, finance, environmental, or public-language lenses.

## Gate

Decision: **open_for_first_public_run_scoping**

Rationale: TARMAC can accept bounded source and review contributions before a
first cited run. Reuse alone does not create engineering, procedure, slot,
route-award, procurement, endorsement, or validation claims.
