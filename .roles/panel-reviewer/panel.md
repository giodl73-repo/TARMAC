# Aviation Peer Panel (illustrative)

Archetype peer reviewers for TARMAC research outputs — distinct from the parliament
(adversarial design review) and the editorial gate (form review). Each is an archetype, not
a named person.

| ID | Reviewer archetype | Expertise |
|---|---|---|
| R-P1 | Air-transport planner | Airport/airspace system planning, hub networks |
| R-P2 | Network scientist | Air-route topology, graph algorithms, resilience |
| R-P3 | Aviation economist | Route economics, competition, benefit-cost |
| R-P4 | Operations researcher | Delay propagation, capacity, ATC flow |
| R-P5 | Aviation policy / equity scholar | Access, EAS, environmental justice |

Assignment guide:

- Topology / resilience method → R-P2, R-P4
- Capacity / delay → R-P4, R-P1
- Route economics / competition → R-P3
- Access / equity / environment → R-P5, R-P1
- Tier/SLA + multi-scale methodology → R-P1, R-P4
